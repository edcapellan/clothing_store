# 💅 NailVerse — Press‑On Nail Store (Flask + Stripe)

NailVerse is a modern, lightweight e‑commerce web application built with **Flask**, featuring:

- 🛍️ Product catalog with categories  
- 🛒 Shopping cart  
- 💳 Stripe PaymentIntent checkout  
- 🛠️ Admin panel (add, edit, delete products)  
- 📦 JSON‑based product storage  
- 🎨 Clean, responsive UI  

This project is fully deployment‑ready for **Render**, **Railway**, **Heroku**, or any WSGI‑compatible host.

---

## 🚀 Features

### **Storefront**
- Browse products by category  
- Add items to cart  
- View and remove cart items  
- Secure Stripe checkout  
- Success & error pages  

### **Admin Panel**
- Password‑protected login  
- Add new products  
- Edit existing products  
- Delete products  
- Products stored in `products.json`  

### **Tech Stack**
- **Flask 2.3+**
- **Stripe API**
- **Gunicorn** (for production)
- **HTML/CSS (no frameworks)**
- **JSON storage**
